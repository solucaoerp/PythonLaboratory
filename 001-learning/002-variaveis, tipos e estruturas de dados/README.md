### Operadores matemáticos

Praticamente, todos os operadores aritméticos funcionam da mesma forma como os conhecemos da matemática elementar.

Operação   | Operador
--------- | ------
adição | +
subtração | -
multiplicação | *
divisão | /

Operadores para exponenciação

Operação   | Operador
--------- | ------
exponenciação | **
